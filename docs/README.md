# Documentation

This folder contains project documentation and visual material for the
Keyence LJ-X8400 custom UI & hardware integration project.

## Contents

- `screenshots/`  
  UI screenshots demonstrating live profile capture, measurements,
  comparison views, and 3D reconstruction.

- `report/`  
  Final project report (PDF) describing system design, implementation,
  and evaluation.

- `presentation/`  
  Final presentation slides used for academic project evaluation.
